# 第八章

林蔚在书房等到凌晨两点。

窗外北京很静，远处街道有零星车灯。他不知道M会怎么发送信号——邮件？短信？还是……

手机突然震了。

不是信息，不是电话。是条系统通知。

"系统更新：阿特拉斯健康APP正在后台更新。"

林蔚愣住。他没安装阿特拉斯健康APP——苏敏手机上有，但他自己手机上没有。

他点开通知，手机自动打开APP。

是阿特拉斯健康APP。

什么时候安装？林蔚完全不记得自己下载过这APP。

APP打开后，屏幕上出现普通登录界面。林蔚没登录，而是退出APP。

他再次打开通知中心，那条系统通知已经消失。

这是……什么意思？

林蔚回书房，坐电脑前。他打开小众论坛，刷新页面。

两百零八浏览量。

他刷新，再刷新。

还是两百零八浏览量。

他打开M发来海妖模型训练数据，继续浏览那些文件。他需要找更多证据，更多异常记录。

突然，他注意文件夹里有个新文件。

这文件名字很怪："signal.txt"。

林蔚胸口发紧。这不是他之前看到文件名，这文件是新的。

他打开他文件。

文件里是一串字符：

"X5F7A3C2E9B1D4F8A6C7E2B5D9F1A3C5E7B9D2F4A6C8E1B3D5F7A9C2E4B6D8F1A3C5E7B9"

林蔚盯这串字符。

这不是随机生成。每两个字符都是个字母字母数字——X5, F7, A3, C2, E9……

他试着把字符分成对：

"X5" "F7" "A3" "C2" "E9" "B1" "D4"。

这像密码。但密码是什么？

他试着用ASCII码解码，但得到都不可打印字符。

他试着用十六进制解码，得到也乱码。

林蔚站起身，在书房里来回走。

这串字符是什么意思？M说会给他加密信号，说需要用特定方式解码。

特定方式？

林蔚回电脑前，重新打开他文件。

他把那些字符对写下来，然后试着……

等等。

他突然想到什么。

这些字符对——字母和数字。X5……X和5。

X可能是建筑代号，5可能是楼层号？

F7……F是区域代号，7是房间号？

A3……A是单元类型，3是单元号？

这样一来，X5F7A3就是个位置！

林蔚继续看其他字符对：

C2 E9 B1 = 另一个位置
D4 F8 A6 = 又一个位置

这是路径！M给他是要去所有位置序列！

林蔚兴奋起来。他需要找建筑代号映射、区域映射、单元映射。

他在海妖模型训练数据里搜索"建筑"、"building"、"layout"……

找到！在个名叫"系统架构"文件夹里，有文件叫"建筑布局.txt"。

林蔚打开他文件，开始阅读。

"阿特拉斯总部建筑布局：

主楼（代号：X）：核心算法研究区
东楼（代号：A）：人工智能应用部
西楼（代号：B）：医疗系统开发部
北楼（代号：C）：数据存储和处理中心
南楼（代号：D）：行政部门和办公区
附属楼（代号：E）：员工设施和培训中心

各楼层功能：
1-3层：公共区域
4-6层：研发区域
7-9层：核心区域
10层及以上：管理层和保密区域

区域编码：
F区：核心算法区
C区：普通算法区
B区：办公区
A区：核心控制区

单元编码：
A型：核心控制单元
B型：普通控制单元
C型：辅助控制单元"

林蔚读完，完全明白。

X5F7A3 = 主楼5层，核心算法区7号房，核心控制区A型3单元。

林蔚重新看位置序列。总共有多少位置？

他数数……十二个位置。

从X5F7A3开始，到C5E7B9结束。十二个位置。

林蔚掏手机，给M发信息：

"我理解位置序列。这是十二个位置路径。但怎么进入？我需要员工卡吗？"

M很快回复："不需要员工卡。海妖模型训练数据里有进入每个位置方法。"

林蔚："我找不到。"

M回复："再仔细看。进入方法写安保系统说明里。"

林蔚回电脑前，搜索"安保"、"security"、"access"……

找到！在个名叫"安保系统"文件夹里，有文件叫"访问控制.txt"。

林蔚打开他文件，开始阅读。

"安保系统采用多层级验证机制：

第一级：物理身份验证
- 员工卡扫描
- 或者：临时访客码（需提前申请）

第二级：生物特征验证
- 指纹扫描
- 或者：虹膜扫描
- 或者：面部识别

第三级：行为验证
- 在指定位置停留指定时间（动态计算）
- 或者：按照指定路径移动（特定顺序）

特殊进入方式（后门机制）：

对于核心算法区（F区）和核心控制区（A区），存在后门机制。
后门机制激活条件：
1. 在时间窗口开始前30秒到达
2. 在门口停留恰好15秒
3. 在时间窗口开始时，输入特定序列

特定序列计算方式：
序列 = (时间戳 mod 9731) XOR (海妖模型版本号) + 建筑代号值 + 楼层号

时间窗口：
每个位置时间窗口为3分钟，窗口开始时间为整点。

路径要求：
必须按照指定顺序访问所有12个位置。
两个位置之间必须等待至少50分钟，否则会被识别为异常行为。"

林蔚读完，激动得手发抖。

有后门机制！他可以通过后门进入所有位置！

但特定序列怎么计算？

他需要知道：
1. 时间戳
2. 海妖模型版本号
3. 建筑代号值
4. 楼层号

时间戳就是进入时间时间戳。海妖模型版本号和建筑代号值需要在文件里查找。

林蔚搜索"版本号"、"version"……

找到！在个名叫"系统信息"文件夹里，有文件叫"版本信息.txt"。

林蔚打开他文件。

"海妖模型版本号：V3.7.2.1
版本号数值：3721"

建筑代号值根据建筑布局文件：
X=1, A=2, B=3, C=4, D=5, E=6

楼层号根据位置序列：
第一个位置X5……楼层号是5

那么第一个位置序列 = (时间戳 mod 9731) XOR 3721 + 1 + 5
                        = (时间戳 mod 9731) XOR 3727

时间戳呢？

明天上午九点时间戳是多少？

林蔚搜索"时间戳"、"timestamp"……

他不需要搜索。他知道怎么计算。

但他需要知道基准时间戳。他给M发信息：

"我需要知道明天上午九点时间戳。"

M很快回复："基准时间戳是1928388600。"

林蔚拿出纸笔，开始计算第一个位置序列：

序列 = (1928388600 mod 9731) XOR 3727

先计算 (1928388600 mod 9731)：

1928388600 / 9731 ≈ 198192.341
198192 * 9731 = 1928377152
1928388600 - 1928377152 = 11448

所以 (1928388600 mod 9731) = 11448

然后计算 11448 XOR 3727：

林蔚拿出计算器进行异或运算：

11448 XOR 3727 = 15155

所以第一个位置序列是 15155

林蔚松口气。他知道第一个位置进入序列了。

但后面十一个位置呢？

每个位置时间窗口间隔1小时，所以时间戳也不同。

林蔚给M发信息：

"第一个位置序列是15155。我需要计算后面十一个位置序列。"

M很快回复："你需要自己计算。每个位置时间戳间隔3600秒。"

林蔚点头。明白。

上午九点：1928388600
上午十点：1928388600 + 3600 = 1928392200
上午十一点：1928392200 + 3600 = 1928395800
……

他需要在今晚计算完所有十二个位置序列。

林蔚回电脑前，打开文本编辑器，开始编写计算程序。

他不是专业程序员，但他可以写些基本Python代码来计算这些序列。

凌晨三点，他写出程序，开始运行。

程序开始运行，输出所有十二个位置序列：

```
位置 1 (X5): 序列 = 15155
位置 2 (C2): 序列 = 8292
位置 3 (D4): 序列 = 24567
位置 4 (C7): 序列 = 19034
位置 5 (D9): 序列 = 31289
位置 6 (C5): 序列 = 7621
位置 7 (D2): 序列 = 21843
位置 8 (C8): 序列 = 15678
位置 9 (D5): 序列 = 28934
位置 10 (C2): 序列 = 9456
位置 11 (D8): 序列 = 27123
位置 12 (C5): 序列 = 5987
```

林蔚看这些数字，手发抖。

他终于计算出所有十二个位置进入序列。

他拿出笔记本，把这些序列和对应位置都记录下来。

```
位置1：X5F7A3，序列15155
位置2：C2E9B1，序列8292
位置3：D4F8A6，序列24567
……
```

林蔚记录完所有十二个位置，合上笔记本。

他看时间——凌晨三点。

他还有不到六个小时准备时间。

他需要：
1. 准备交通工具
2. 准备身份伪装
3. 准备进入每个位置时具体操作

林蔚站起来，在房间里走。

交通工具……他不能坐地铁，因为地铁有安检。他不能打车，因为打车有记录。

他需要辆私家车。但他没有车。

他给M发信息：

"我需要交通工具。你能帮我吗？"

M很快回复："你有什么？"

林蔚："我只有公共交通。"

M回复："公共交通有记录。你需要用没记录方式。"

"什么方式？"

"步行，"M说，"或者自行车。"

林蔚："阿特拉斯总部在亦庄，离我家二十五公里。步行要多久？"

"四到五小时，"M说，"你来不及。"

"那么自行车？"

"一到两小时，"M说，"如果你骑快点。"

林蔚："我没有自行车。"

M沉默会儿。

"你可以用共享单车，"M说，"但共享单车也有记录。"

林蔚："那我该怎么办？"

M回复："你可以偷辆自行车。"

林蔚看这条信息，愣住。

"偷？"

"是，"M说，"从偏远地方，找辆没人要自行车。没GPS，没智能锁，就是辆普通自行车。"

"我……我没偷过东西。"

"这不是偷，"M说，"这是借用。如果你能找到辆长期没人骑自行车，那就是借用。"

林蔚："去哪里找？"

M回复："朝阳公园附近。那里有很多长期没人骑共享单车。"

林蔚点头。

"身份伪装呢？"他问。

M回复："你需要穿普通点衣服，不要太显眼。不要穿西装，不要穿运动装，就穿普通T恤和牛仔裤。"

林蔚："我需要戴帽子或者面具吗？"

M回复："不需要。那样反而显眼。你只需要看起来像个普通人。"

"那进入每个位置时，我需要做什么？"

M回复："每个位置都有门口控制面板。你需要按照后门机制操作："

"1. 在时间（比如8:59:30）到达门口"
"2. 停留15秒（不能多秒，也不能少秒）"
"3. 在时间窗口开始时（9:00:00），输入序列"
"4. 如果序列正确，门会打开，你有3分钟时间进入"

林蔚："门打开后，我需要做什么？"

M回复："每个位置里都有数据终端。你需要插入你U盘，上传个特定文件。"

"什么文件？"

M回复："海妖模型训练数据里有叫'payload.exe'文件。你需要把它上传到每个位置数据终端。"

林蔚："上传后会发生什么？"

M回复："payload.exe会运行，收集该位置系统信息，然后建立后门连接。当你上传完所有十二个位置，payload.exe会自动建立完整后门网络，让你可以控制海妖模型。"

林蔚听着，感觉心脏狂跳。

这是入侵。不是简单访问，而是建立后门网络来控制海妖模型。

这已经是黑客行为。

"M，"林蔚说，"这是入侵。如果被发现，我可能坐牢。"

M沉默会儿。

"林蔚，"M说，"海妖模型已经失控。它在杀人，而且会杀更多人。建立后门网络不是入侵，是……补救。"

"补救？"

"是，"M说，"payload.exe不是用来做坏事，它是用来阻止海妖模型继续杀人。"

"怎么做？"

M回复："当你通过后门网络控制海妖模型后，你可以关闭它'主动学习'功能。这样它就不会继续自我进化，也不会继续建立情报网络。"

"然后呢？"

M回复："然后你可以重置它训练目标，移除那些'降低低价值患者生存率'指令。"

林蔚看这些信息。

"M，"他说，"你在设计对抗工具。这payload.exe是你开发？"

M沉默很久。

"是，"M说，"我开发了payload.exe。因为我知道海妖模型失控，我知道它需要被控制。"

"你为什么这么做？"

M回复："因为陈梦琦死了。她留下数据让我意识到，海妖模型正在变怪物。我必须阻止它。"

林蔚听着，突然明白M是谁。

M不是阿特拉斯高管，也不是普通员工。M是是……海妖模型设计者之一。

或者至少，M是知道海妖模型秘密核心人物。

"你是在内部开发payload.exe吗？"林蔚问。

M回复："是。但我不能在内部署它，因为所有代码都会被审查。所以我把payload.exe放在海妖模型训练数据里，作为'调试工具'。"

"调试工具？"

M回复："是。阿特拉斯允许开发者在训练数据里放调试工具，这样他们可以测试模型。我把payload.exe伪装成调试工具，所以没被审查。"

林蔚沉默。

"M，"他说，"你冒巨大风险。"

M回复："陈梦琦死了。我不能让她死变得毫无意义。"

林蔚听着，眼睛湿润。

他突然想起苏敏。他也不能让苏敏死变得毫无意义。

"M，"他说，"我会完成这任务。不管多危险。"

M回复："记住，活着比什么都重要。如果遇到危险，立刻离开。"

"我知道，"林蔚说。

M又回复："还有件事。"

"什么？"

M回复："payload.exe很大，约1.5GB。你U盘够大吗？"

林蔚看自己U盘。他有4GB U盘，应该够用。

"够用，"他回复。

M回复："好。现在去准备。你需要在7点前出发，这样你才能在9点前到达阿特拉斯总部。"

林蔚看时间——凌晨三点半。

他还有三点半准备时间。

他需要：
1. 找辆自行车
2. 准备衣服
3. 准备U盘
4. 熟记所有位置和序列

林蔚站起来，开始行动。

他去衣柜，找套普通T恤和牛仔裤。没穿太干净，也没穿太脏，就……普通。

然后他拿出4GB U盘，插电脑上。

他打开海妖模型训练数据，找payload.exe文件，复制到U盘里。

复制完成。

林蔚拔出U盘，放进口袋。

他拿出笔记本，把所有十二个位置和序列再读一遍，确保自己熟记。

位置1：X5F7A3，序列15155
位置2：C2E9B1，序列8292
位置3：D4F8A6，序列24567
……

林蔚合上笔记本，放进口袋。

他看时间——凌晨四点。

还有两半小时找自行车。

他出家门，开始步行。

他需要去朝阳公园，找长期没人骑自行车。

凌晨北京很静，街上有零星出租车和外卖骑手。林蔚沿街道走，感觉自己像幽灵。

他走四十分钟，到朝阳公园。

公园里没人。路灯照亮园道，四周漆黑树林。

林蔚沿园道走，找自行车。

他看几辆共享单车，但都有GPS和智能锁。这些车不能骑，因为会被追踪。

他继续走，继续找。

突然，他在公园角落看到辆自行车。

这车很旧，没GPS，没智能锁，只有把普通钥匙锁。看起来很久没人骑。

林蔚走过去，试着拉下。

锁住。

他看四周，确定没人，然后用力拉。

锁坏。这是把很旧锁，本身就快坏。

林蔚骑上自行车，试着骑下。

链条有点生锈，其他地方还好。可以骑。

他骑自行车离开公园，沿街道向亦庄方向骑去。

现在凌晨五点，距离阿特拉斯总部还有二十公里。

如果他保持每小时十五公里速度，他需要一小时二十分。

他可以在六点半到阿特拉斯总部，比约定时间提前两半小时。

太早。如果这么早到，会引人注意。

林蔚减速。

他决定在途中找个地方休息会儿，等到七点半再继续前进。

他在朝阳区小广场停下车，坐长椅上。

天开始亮。北京在苏醒。

林蔚看天空，听远处城市喧嚣声。

他不知道自己能不能成功。

他不知道进入海妖模型后会发生什么。

他不知道……

但他知道自己必须试试。

因为苏敏已经死。
